# O-Watch Beta — שעון התרעות חכם

מערכת התרעות לבישה מבוססת BLE ו-ESP32 לקבלת התרעות בזמן אמת באמצעות אפליקציית Android.

---

## ⚠️ גרסת Beta

המערכת נמצאת בשלב אבטיפוס ופיתוח פעיל.

ייתכנו:
- תקלות תקשורת BLE
- חוסר יציבות זמני
- שינויי חומרה ותוכנה
- מגבלות פונקציונליות

---

## ✨ יכולות עיקריות

- 📳 מנוע רטט מובנה
- 🌈 חיווי צבעים באמצעות NeoPixel LED
- 📱 חיבור Bluetooth Low Energy
- 🚨 תמיכה בהתראות פיקוד העורף
- 🛰️ שליטה דרך אפליקציית Android
- ⚡ אבטיפוס קומפקטי לשימוש יומיומי

---
